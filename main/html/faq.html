Frequently Asked Questions

Overview

Automatic Formatting

When a test sheet is generated, the following changes are performed automatically:

  • Extra spaces are removed
  • Capitalization is fixed
  • Phone numbers are formatted
  • Postal codes are formatted
  • LSS IDs are formatted and added in parentheses (when applicable)
  • Fields are formatted together

Usage

Home Page

The home page allows you to select which test sheet you wish to use and where your data will come from.

Test sheets are separated into four categories:

  • Lifesaving & Lifeguarding (Bronzes & NL)
  • First Aid (Emergency & Standard First Aid)
  • Leadership (Master Sheets)
  • Other (Other Streams)

Press the "Use" button next to an entry in these lists to proceed.

You will be given the choice of which source your candidate data will come from.

Select a source and follow the prompts to be directed to the matching page.

Matching Page

The matching page allows you to see the candidate responses that have been submitted, pick which ones to include, filter responses and edit responses before generating a test sheet.

The row at the top of the screen allows you to match questions to test sheet fields.

The left edge of the screen allows you to select which responses to include.

The bottom of the screen provides information on which test sheet you have selected and options to Cancel or Continue to sheet generation.

If you want to manually add a response, an Add Response button can be found below the last entry.

If you want to add a field entry to every response, a + button can be found in the top row at the right of the last column.

Selection

You can choose which responses are included on a test sheet.

A count of how many responses are to be included on the test sheet is displayed in the bottom right of the screen.

You can include or exclude a response using the checkbox on the left next to each response.

Filtering

When 10 or more responses exist, a filter button will exist in the top row for each column.

The filter button for a column will be highlighted when a condition has been set.

The filter menu allows you to only see responses that meet the specified criteria.

Multiple filters can be active at the same time. All filters must be satisfied for an entry to be shown.

Field Matching

The top row of the matching screen is composed of multiple dropdowns that allow you to assign a question to a field (Name, Address, Date of Birth, etc) on the test sheet.

For example, if you set a dropdown to "Name", the information for each response in that column will fill the "Name" field on the test sheet.

The dropdowns will be set automatically when you first start. They are automatically assigned to their most likely option when loading a source.

This automatic process is based on the titles of each question as well as the response content.

If a dropdown is left blank, the response data in that column will not be included on the test sheet).

The same field cannot be set twice, so you cannot assign the "Name" field to two separate columns.

Editing Data

Every box containing candidate data can be manually edited. Simply click the box and change the information.

See Automatic Formatting for information on what edits will be performed automatically when generating sheets.

It may be required to perform some minor edits if incorrect information is supplied by candidates.

Age Prerequisites

For any courses that have an age prerequisite, the Date of Birth field will be used to calculate the candidate's current age.

If the candidate is younger than the age prerequisite, their Date of Birth will be highlighted and a notice will appear at the bottom of the screen.

The notice will contain information on whether the age prerequisite has alternatives or not.

Age is calculated based on the present day. If the exam date is in the future, check any highlighted candidates to ensure they will meet the age prerequisite by the exam date.


For more information on age prerequisites, consult the Lifesaving Society website.

Information Page

After pressing the 'Confirm' button, you will be asked to fill in the course information.

This section is not required, but does offer three advantages:

  • With larger classes, the information will be filled in on all test sheets at the same time
  • If you are filling out multiple test sheets (such as a combined Bronze Medallion and Emergency First Aid), your information is saved and a "Fill in From Previous Sheet" button will appear after the first time.
  • Your browser autofill functionality is available, making it quicker to fill in your information.

When you are done filling in the course information (or wish to skip it), press the "Generate Test Sheet" button. After a brief moment, your test sheet(s) will be available for download.

Sources

Google Forms

Google Forms can be used as a source, allowing for quick collection of candidate information.

When you select Google Forms, you will be asked to connect to your Google account.

Only the form you select is readable by this site, and all data is processed locally on your computer.

CSV Files

Comma-Separated Value () files can used as a source.

These files are a common export format from many survey tools.

The first row should contain header values.

Other Sources

Other data sources can be requested by clicking the 'Contact Us' link on the home page.

Microsoft Forms are not available due to a technical limitation on the part of Microsoft.

Setup

When setting up your information collection form, these are some best practices to keep in mind:

  • Sections or dividers have no impact. You can use any navigation features provided without issue.
  • The title of questions are used to help match them correctly. Try to include the name of the field in the title (Name, Date of Birth, Address, etc).
  • The date format used is YYYY-MM-DD. Most tools convert to this format automatically, but manual data sources (like registration reports) may need to be manually edited.
  • It can be helpful to include a dropdown question asking what course a candidate is taking. This makes filtering the data easier when selected entries.
  • Some tools allow for input validation (such as ensuring a value is an email address, phone number, a specific length, etc). Using these options can help ensure you receive the proper information.

If you intend to use prerequisite information fields:

  • Prerequisite fields are detected based on their names. They must include the name of the prerequisite must be included in the title. Example: "Where did you take your Bronze Cross?
  • Date and Location fields for each prerequisite are matched automatically based on the response content. You do not need to include any specific words to differentiate them.
  • If you have the same prerequisite question multiple times (i.e. Bronze Cross for National Lifeguard and Leadership), you may need to manually pick the correct data.
